# Plan 09-01: Write Firestore Security Rules

## Goal

Write a complete `firestore.rules` file enforcing role-based access for all collections: `users`, `sites`, `site_users`, `transactions`, `audit_logs`. Add the `firestore` target to `firebase.json`.

## Context

- Firebase project ID: `shreegirirajsteel-a758a`
- Auth method: Google Sign-In — `request.auth.uid` matches user UID
- Role field: `users/{uid}.role` — string values: `"superAdmin"`, `"admin"`, `"partner"`
- `isActive`: users with `isActive == false` should not be blocked at Firestore level (blocked at app level by auth guard); rules focus on role-based access, not active status
- Flat root `transactions` collection (not subcollection) — `createdByUserId` field links to user
- `site_users` join table: `siteId` + `userId` — determines partner-site assignments

<action>
Create `firestore.rules` at project root with the following complete content:

```
rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {

    // ─── Helper Functions ─────────────────────────────────────────────

    // Check if user is authenticated
    function isAuthed() {
      return request.auth != null;
    }

    // Get the caller's user document
    function getUserData() {
      return get(/databases/$(database)/documents/users/$(request.auth.uid)).data;
    }

    // Check caller's role
    function hasRole(role) {
      return isAuthed() && getUserData().role == role;
    }

    function isPartner() {
      return hasRole('partner');
    }

    function isAdmin() {
      return hasRole('admin');
    }

    function isSuperAdmin() {
      return hasRole('superAdmin');
    }

    function isAdminOrAbove() {
      return isAdmin() || isSuperAdmin();
    }

    // Check if partner is assigned to a site
    function isAssignedToSite(siteId) {
      return isAuthed() && exists(
        /databases/$(database)/documents/site_users/$(siteId + '_' + request.auth.uid)
      ) == true ||
      // Fallback: query site_users by siteId + userId (compound doc path not always used)
      isAuthed() && existsAfter(
        /databases/$(database)/documents/site_users/$(siteId + '_' + request.auth.uid)
      );
    }

    // ─── users collection ─────────────────────────────────────────────
    // Partners can read their own doc.
    // Admins and Super Admins can read all user docs.
    // Super Admin can write any user doc.
    // Users can update their own location fields only.

    match /users/{userId} {
      allow read: if isAuthed() && (
        request.auth.uid == userId ||
        isAdminOrAbove()
      );

      // Super admin can create/update any user
      allow write: if isSuperAdmin();

      // Any authenticated user can update their own location fields
      allow update: if isAuthed() &&
        request.auth.uid == userId &&
        request.resource.data.diff(resource.data).affectedKeys()
          .hasOnly(['lastLatitude', 'lastLongitude', 'lastLocationAt']);
    }

    // ─── sites collection ─────────────────────────────────────────────
    // Partners can read sites (they need the site name for transactions).
    // Admins/Super Admins can read and write.

    match /sites/{siteId} {
      allow read: if isAuthed();
      allow write: if isAdminOrAbove();
    }

    // ─── site_users collection ────────────────────────────────────────
    // Partners can read their own site_user entries.
    // Admins can read all, write (assign/remove partners).

    match /site_users/{siteUserId} {
      allow read: if isAuthed() && (
        resource.data.userId == request.auth.uid ||
        isAdminOrAbove()
      );
      allow write: if isAdminOrAbove();
    }

    // ─── transactions collection ──────────────────────────────────────
    // Partners can create transactions for sites they are assigned to.
    // Partners can read only their own transactions.
    // Admins can read/create/update/delete all transactions.

    match /transactions/{transactionId} {
      // Read: own transactions for partners; all for admins
      allow read: if isAuthed() && (
        resource.data.createdByUserId == request.auth.uid ||
        isAdminOrAbove()
      );

      // Create: partners create their own; admins create any
      allow create: if isAuthed() && (
        (isPartner() && request.resource.data.createdByUserId == request.auth.uid) ||
        isAdminOrAbove()
      );

      // Update/Delete: admins and super admins only
      allow update, delete: if isAdminOrAbove();
    }

    // ─── audit_logs collection ────────────────────────────────────────
    // Any authenticated user can create audit log entries.
    // Nobody can read, update, or delete audit logs from the client.

    match /audit_logs/{logId} {
      allow create: if isAuthed();
      allow read, update, delete: if false;
    }

    // ─── Deny all other collections ───────────────────────────────────
    match /{document=**} {
      allow read, write: if false;
    }

  }
}
```
